# Aplicaciones interactivas del curso

Herramientas web creadas para practicar los temas del curso. No requieren instalación: se abren directamente en el navegador.

| Aplicación | Qué hace | Acompaña a | Enlace |
|------------|----------|------------|--------|
| **Tablas de Verdad** | Genera la tabla de verdad de una expresión lógica: conectores AND, OR, NOT, implicación (`->`) y bicondicional (`<->`), con variables `p`, `q`, … | Sesiones 1–2 (lógica proposicional) | [Abrir aplicación](https://true-table-prope-1--true-table-prope.us-east4.hosted.app/) |

Las aplicaciones nuevas se irán añadiendo a esta tabla a lo largo del curso. Si más adelante el código fuente de una aplicación se publica en este repositorio, vivirá en una subcarpeta propia: `aplicaciones/<nombre-app>/`.
